Upload a Bank Statement


1. Download your electronic statement

Visit your online banking website and download an electronic statement for your bank or credit card account in any of these file types:

.Csv Csv file
.Pdf Pdf file
.xlsx Excel file
.docx Word file
2. Upload your statement to Wave
Once you have your electronic statement downloaded to your computer, upload it to Wave using the form below.

Statement*

Payment Account*

Select the account for which you're uploading a statement, or add a new account.


3.Manage your transactions
After you upload the file, Wave will add the transactions in the file to the Imported Transactions page.
